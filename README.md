A funny trivia game.

# Instructions to build locally

#### Clone the repo

`git clone https://github.com/barhoring/trivia-app.git`

#### Install dependencies

`cd trivia-app`
`npm install`

#### Play

`npm start`

Go to [http://localhost:3000/](http://localhost:3000/)
